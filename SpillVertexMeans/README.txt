This directory contains npz files labeled by spill. Each file contains 7 0-dimensional arrays stored in npy files
The 0th array contains the spill ID
The 1st, 2nd, and 3rd arrays contain the mean X, Y, and Z vertex positions over all events in the spill respectively
The 4th, 5th, and 6th arrays contain the standard deviations of X, Y, and Z vertex positions respectively
